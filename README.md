# My dotfiles

This repository containes some of the files I currently use in my development environment:

- .vimrc
- .zshrc
- .hushlogin

The `.vimrc` is pretty self-explanatory and needs a symbolic link to the home directory.  
The `.zshrc` needs to be copied because otherwise it does too many symbolic links.  
The `.hushlogin` file disables the welcome message while logging.
